# objectCache - 一个用golang实现的本地对象缓存库。


## 特性
1、支持最多存储对象个数设置（此个数实际是一个参考值，淘汰算法会尽量满足）。

2、支持高并发。

3、支持过期时间设置。

4、使用同时兼顾访问频率、访问稳定性的淘汰算法进行数据淘汰。

5、没有定期扫描所有对象的高开销。

## 性能
高并发下，读写速率、对GC的压力(实际运行趋于0)、内存的额外开销、对CPU的占用都趋于map，优于sync.map。

读写速率参考cache_benchmark_test.go
